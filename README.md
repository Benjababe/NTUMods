# Pre-requisites

Ensure you have a `.env` file in the project root with the following properties set

```
# postgres database name
PG_DB=

# postgres username
PG_USER=

# postgres user password
PG_PASSWORD=

# postgres host IP
PG_HOST=

# postgres exposed port
PG_PORT=
```
